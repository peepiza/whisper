## О проекте

**"Шёпот"** — это 2D-платформер с уникальной механикой управления через микрофон. Вместо традиционной клавиатуры игрок использует свой голос, чтобы освещать путь в тёмной пещере. Чем громче звук — тем больше радиус освещения, но тем быстрее заполняется шкала опасности. Слишком громкий крик или долгий шум привлекают внимание тьмы и ведут к проигрышу.

### Особенности

- **Аудиоуправление** — громкость голоса влияет на видимость платформ
- **Динамическое освещение** — радиус свечения зависит от громкости
- **Рекорды времени** — сохраняется лучшее время прохождения уровня
- **PWA** — можно установить на рабочий стол как нативное приложение
- **Атмосферная графика** — тёмная пещера, светящийся персонаж, пульсирующие движущиеся платформы
- **Три варианта поражения** — с разными атмосферными сообщениями

## Управление

| Клавиша | Действие |
|---------|----------|
| ← → | Движение влево / вправо |
| ↑ | Прыжок |
| ESC | Пауза |

### Микрофон

- **Шепчите** — платформы становятся видны в небольшом радиусе
- **Говорите громче** — радиус освещения увеличивается
- **Крикните слишком громко** → проигрыш *"Тьма услышала тебя..."*
- **Издавайте звук слишком долго** → шкала шума переполняется → проигрыш

## Уровень

Уровень представляет собой пещеру длиной 3400 пикселей. Игроку предстоит преодолеть:

- Статические платформы (тёмно-голубые)
- Движущиеся платформы (ярко-голубые, всегда видны)
- Опасные шипы (ярко-синие с фиолетовой окантовкой)

### Сообщения при проигрыше

| Причина | Сообщение |
|---------|-----------|
| Слишком громко | *"Тьма услышала тебя..."* |
| Шипы | *"Пещера небезопасна..."* |
| Падение в бездну | *"Пещера оказалась глубока..."* |

### Победа

При достижении правого края уровня (x > 3200) игрок побеждает. Если время прохождения побило рекорд — появится уведомление.
